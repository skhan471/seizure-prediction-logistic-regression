\# Seizure Prediction using Logistic Regression



Machine learning project for epileptic seizure prediction using Logistic Regression and preprocessing pipelines.



\## Project Features

\- EEG-inspired seizure prediction workflow

\- Logistic Regression models

\- L1, L2, and Elastic Net regularization

\- SMOTE and class imbalance handling

\- Overfitting vs underfitting analysis

\- IEEE-style LaTeX report

\- Presentation slides
